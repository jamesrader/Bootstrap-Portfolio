Methodology Checklist (JavaScript, PHP, MySQL)

This is a database of all the possible problems a player could have with each of his/her tennis strokes. A coach can mark the issues in one of 3 languages (English, Spanish or Chinese). When finished the coach can input the player's name, a date for the report and an output language among the same 3 languages.

A formatted report is created as a Word document, ready for adding a cover photo and sending to the player/parents. The report which looks like substantial time was spent in building, can be generated in just minutes.

This was accomplished using JavaScript, PHP and MySQL.
