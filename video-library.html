Video Library (JavaScript, PHP, MySQL)

This is a video library consisting of over 3,500 tennis video drills stored on Amazon S3. It was originally written in Flash/ActionScript 3, and the data of each drill was contained in an XML file.

I updated this app by completely re-writing it in JavaScript. I converted the XML file to a MySQL database. I used an XSLT and Excel to convert the data for import into the database.

The dropdowns allow for filtering the drills are all populated dynamically, of course. Access to the database is accomplished through PHP.
