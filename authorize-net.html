Custom Authorize.net Sales Stats

We needed a method to see the sales stats from Authorize.net in a convenient, customized way. I was able to write a PHP script to access the Authorize.net API. I used a Cron job to run the script every night. This allowed me to get the information for each day's batch of transactions. It was returned in the form of XML data. After parsing the XML data, I used Flash (this was originally built in 2010) to present the information on the Drupal site.
