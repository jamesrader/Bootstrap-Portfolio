Speed Tracker/Speed Viewer (C# and MySQL)

This is actually a combination of two desktop apps (Speed Tracker and Speed Viewer) that I wrote in C#.

Speed Tracker takes serial input from a radar gun through an RS232 to USB converter. A user can then mark on a tennis court diagram where the ball landed. This data is stored in a MySQL database, and simple stats are displayed on the screen as the data is collected.

The companion program Speed Viewer allows for viewing of historical data in detailed scatter chart and table views for in-depth analysis.
